# 리뷰

- `기업별 재무 조회`와 `조건 검색`을 위아래 나열이 아닌 탭 인터페이스로 분리해 기능 전환 흐름을 명확히 함
- 입력 라벨은 기존처럼 항상 표시되도록 유지하고, 탭 스타일을 현재 디자인 톤에 맞게 조정함
- `python3 -B -m py_compile app.py`로 문법 검증 완료
- `pytest -q`는 로컬 환경에 `pytest` 명령이 없어 실행하지 못함
