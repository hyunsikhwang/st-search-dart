# 리뷰

- 입력 폼을 단일 열 구조로 단순화하고 라벨을 `visible`로 명시해 모바일에서도 `회사명`, `기준 연월`, `직전 분기 수`, `최소 영업이익률`이 보이도록 보완함
- DB 저장 현황 카드 폭을 넓히고 보조 문구를 제거해 좌우 카드의 줄바꿈과 높이 증가를 줄이도록 조정함
- `python3 -B -m py_compile app.py`로 문법 검증 완료
- `pytest -q`는 로컬 환경에 `pytest` 명령이 없어 실행하지 못함
