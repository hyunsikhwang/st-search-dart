# 리뷰

- 기준연월 항목을 여러 pill이 아닌 단일 행 텍스트로 렌더링해 모바일에서 카드 높이가 늘어나지 않도록 수정함
- 모바일에서는 저장 분포 카드에 `max-height`를 적용하고 내부 텍스트는 가로 스크롤로 처리함
- `python3 -B -m py_compile app.py`로 문법 검증 완료
- `pytest -q`는 로컬 환경에 `pytest` 명령이 없어 실행하지 못함
